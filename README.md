# RustIOT
## 你的工厂离 4.0 只差一个盒子
### 基于 Rust 构建的轻量、高性能、旁路由式工业边缘网关，实现 Modbus ↔ MQTT ↔ WebSocket 全协议互通，支持 RK 系列 ARM 芯片与 Armbian 系统原生部署。

RustIot 是一款纯软件定义的工业物联网边缘网关，以 Rust 保证高稳定性、低延迟、低资源占用，采用旁路由模式零侵入接入现有工厂网络，无需改造设备、无需更改拓扑，即可快速实现设备数据采集、协议转换、本地可视化与数据上云。
Rust 原生高性能：无 GC、内存安全、7×24 小时稳定运行
旁路由零侵入部署：即插即用，不影响现有网络结构
多协议双向互通
Modbus TCP / RTU 设备采集
Modbus ↔ MQTT 双向转发
WebSocket 实时数据推送
边缘自治运行：断网缓存、自动重连、本地自治
跨平台支持：ARM /x86 全架构兼容
原生支持 Armbian：RK3568 / RK3588 开箱即用
内置 Web 管理面板：可视化配置、实时监控、日志查看

支持协议
Modbus TCP / RTU（主站）
MQTT 3.1.1 / 5.0
WebSocket（RFC 6455）
HTTP 管理接口

RK3568 / RK3588（推荐，Armbian）
树莓派 4B / 5B
x86_64 软路由 / 工控机
所有 Linux 系 ARM 设备

└─────────────────────┘
   ↙            ↓

              Web 前端（WebSocket 实时展示）


适用场景
工厂 PLC / 传感器数据采集上云
老旧设备轻量化工业 4.0 改造
本地网页实时监控（无云端依赖）
轻量级协议转换与数据转发
低成本替代传统商业工业网关
